### Install

1. Download zip from ```https://github.com/jsan3386/avatar ``` <br />
2. Install Addon in Blender
3. Download [cloth textures](https://cv.iri.upc-csic.es/Dataset/avatar/textures.zip) and extract files in avatar folder
